香菇街html+css布局浅谈

香菇街的案例参照 表严肃 的视频 ，其项目代码见 github.com/biaoyansu/11.x

先说说香菇街的布局

主区

导航栏——.top-nav

搜索栏——.header

主促销区——.main-promote

次展示栏——.cat-promote

页脚栏——.footer

通篇以div+class的布局方式。然后提前设定好百分比的分块。再实际布局中再加以利用，width百分比的好处是可以在浏览器大小改变时能够相应收缩放大，但是对于图片较多的电商网站则会造成图片压缩，所以像淘宝等电商网站通常固定width，不改变其大小。香菇街的案例仍采用百分比的模式。

总的CSS布局

这里强调一下box-sizing：border-box，全局都应用边框盒子模型，可以减少因为margin引起的问题，同时不影响子div灵活设置margin。

这里给body设置全局的文字大小，颜色和背景。删除了a链接的下划线，设置图片的max-width:100%是防止图片过大，其最大宽度为父元素宽度的100%。强调为方便看布局，可设置其背景颜色。

这里每个大的分区都在.container的div下，这样可以保证全站上下格式一致，而.container设置了最大宽度，块状元素，居中。这个很重要！
