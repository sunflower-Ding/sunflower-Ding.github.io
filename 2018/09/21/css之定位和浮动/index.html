css之定位和浮动

在学习CSS布局中，一直深受position和float的困惑，在学习http://zh.learnlayout.com/position.html后略有收获。

position属性

值static

static是默认值，一个static元素表示它不会被positioned。

值relative

这个值非常重要，在一个相对定位（position 属性的值为relative）的元素上设置top、right、bottom和left属性，会使这个元素偏离其正常的位置。其他的元素的位置则不会受该元素的影响发生位置改变来弥补它偏离后剩下的空隙。

值fixed

这个比较好理解，一个固定定位（position属性的值为fixed)元素会相对于视窗来定位。即便页面滚动，它还是停留在原来的位置。

这个固定定位元素脱离了文档流，不会保留它原本在页面应有的空隙，就像是图层，固定飘在上面。（但是移动浏览器对fixed的支持很差）。

值absolute

这个值刚开始比较难理解，不过知道相对论的就很容易理解了。absolute需要一个背景（也可以理解为画布或图层吧），这个背景就是离他最近的“positioned”祖先元素。absolute是在这个背景上绝对定位，也就是说它是相对于这个背景固定的，当然如果这个背景动，它也会动。你就把它当作背景上固定的一副画。

如果绝对定位（position属性的值为absolute)的元素没有“positioned”祖先元素，那么它是相对于文档的body元素，并且会随着页面滚动而移动。在这里，一个“positioned”元素是指position值不是static的元素。

因此我们常见到，想要定位一个元素绝对定位，首先要让他的祖先元素“positioned”，可以让它仅仅“relative”

注意：祖先元素中其他普通元素的margin的使用，可以为绝对定位元素预留一定的空间。

浮动float

clear属性被用来控制浮动，在同一个容器中的A和B，当A浮动left时，B会包围A，使用clear可以使B移动到浮动元素A的下面，即设置B{clear:left;}。
